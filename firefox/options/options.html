KeeForm is an easy-to-use and fast form filler for KeePass. It automatically fills in your username and password fields with a simple double-click in KeePass.

How form filling works

  1. Start KeePass and double-click on a URL.
  2. KeePass passes the URL, username, and password to KeeForm.
  3. KeeForm opens the URL in the browser.
  4. KeeForm automatically fills in the username and password fields.
  5. You review the filled fields and submit the form yourself.

KeeForm uses a smart algorithm that handles standard login forms, forms with hidden password fields, shadow DOM, and many modern single-page application frameworks.

If form filling does not work

Some pages load slowly or build their forms dynamically. If the fields were not filled:

  1. Click the KeeForm icon in the browser toolbar to retry. The icon turns green when a retry is available.
  2. Click into the username or password field first, then click the KeeForm icon. Some modern frameworks (React, Angular, Flutter) only initialize their input fields when the user interacts with them.
  3. For modal/popup login forms: open the modal first, then click the KeeForm icon.
  4. For two-step logins (email on one page, password on the next): click the icon on the password page.

Icon meaning

  • Green badge — KeeForm is connected to KeePass and ready.
  • Red badge — KeeForm cannot connect. Check that the KeeForm Windows application is running and not blocked by antivirus.
  • Green icon (tab) — A retry is available. Click to retry form filling.

Installation

  1. Download the KeeForm Windows application from keeform.org and follow the guided steps.
  2. Install this browser extension from the KeeForm website.

Privacy

KeeForm never has access to your KeePass database. It only receives the username and password for the specific entry you double-clicked, uses it to fill the form, and discards it. No data is stored or transmitted externally.
