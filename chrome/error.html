KeeForm cannot continue until the Windows host application is updated. After installing, restart the browser to reconnect.

What's in this update

  • Complete rewrite of the launcher and Windows host application.
  • AutoIt launcher replaced by a native Go application — faster, smaller, no runtime dependency.
  • DPAPI encryption used for the data file.
  • Minimum extension version check added to the host.
  • 32-bit and 64-bit versions included.
  • KeePass folder pre-filled when installed in a standard location.
  • Silent/automated installation support.
  • Automatically installs the correct version for your Windows.
