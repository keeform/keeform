Security advice: Confirm that you see a button with a puzzle piece icon labeled "KeeForm" in your address bar before downloading and installing anything.

What's new in KeeForm 5.0.1

Release Notes:

  • Complete rewrite of the extension, launcher, and Windows host application.
  • Several form-filling improvements for modern single-page applications.
  • Version check — an error page is shown if the Windows application is outdated.
  • Lighter, faster options page — UIkit and jQuery removed.
  • New lock-shaped icon design.
  • Orange icon indicates when retry filling is available.
  • Gitgub repo available: github.com/keeform/keeform

Did you know?

  • If fields were not filled, click the icon in the browser toolbar to retry.
  • For two-step logins (email first, then password): click the orange icon again on the password page.
  • For modal/popup forms: open the modal first, then click the orange icon.
  • For stubborn forms: click into the username or password field first, then click the orange icon.
  • When the icon turns blue, login data is erased and retry is no longer available.
  • You can select multiple entries in KeePass and trigger form filling for all of them at once using Ctrl+U.
