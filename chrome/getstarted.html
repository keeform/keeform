Welcome to KeeForm

KeeForm requires a Windows application to work. Please download and install it to get started.

Download the KeeForm Windows installer

How KeeForm works

KeeForm is triggered from KeePass — not from the browser. You double-click a URL entry in KeePass, and KeeForm opens that page and fills in your username and password automatically. It does not search KeePass for matching credentials.

When everything is connected and ready, the KeeForm badge gets the green light.

For more information, visit keeform.github.io/pages.
